About MindDeck Flashcards

A simple, privacy-first flashcard application built to help you learn faster and more effectively.

Project Overview

MindDeck Flashcards was created to practice building web applications while studying at the same time. It's a minimal, client-side flashcard app that prioritizes simplicity and privacy—all your data stays in your browser.

Features

Create & Manage Decks

Organize your flashcards into custom decks with names and descriptions. Create unlimited decks for different subjects or topics.

Study Mode

Review cards with an intuitive study interface. Use keyboard shortcuts (Space to flip, arrows to navigate) for efficient studying. Track your progress with visual indicators.

Export & Import

Save your decks as JSON files for backup or sharing. Import decks from other sources to expand your collection.

Privacy First

All data is stored locally in your browser using localStorage. No accounts, no servers, no tracking—just your flashcards.

Undo Support

Accidentally deleted a deck or card? Use the undo feature to restore it quickly.

Clean Interface

Simple and modern design that focuses on your content without distractions.

Technologies

This project is built with:

  • HTML5 — Semantic markup and accessibility
  • CSS3 — Modern styling with CSS variables and responsive design
  • Vanilla JavaScript — No frameworks, just pure JavaScript
  • Express.js — Optional server for future sync features
  • localStorage API — Client-side data persistence

How to Use

  1. Create a deck: Click "Create a deck" and give it a name and optional description.
  2. Add cards: Open a deck and add flashcards with questions on the front and answers on the back.
  3. Study: Click "Study" to enter study mode. Use Space/Enter to flip cards, arrow keys to navigate, and Escape to exit.
  4. Export: Save your decks as JSON files for backup or sharing.
  5. Import: Load previously exported decks to restore or merge collections.

Running Locally

To run this project locally, serve the files from the project root:

python3 -m http.server 8000
# Then open http://localhost:8000 in your browser

For the optional server sync features:

npm install
npm start
# Server runs on http://localhost:3000

Data Storage

All flashcard data is stored in your browser's localStorage under the key wwwf_decks_v1. This means:

  • Your data never leaves your device
  • No account or login required
  • Data persists between sessions
  • You can export/import to back up or transfer data
